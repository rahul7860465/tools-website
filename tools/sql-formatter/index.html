SQL Formatter

Developer Tools

Paste SQL and format it for readability.


            

          


          

            
            
            
            
          

        

      


      

        
How to use this tool

        
Paste SQL, click Format, then copy the result.

        
Example input

        
select id,name from users where active=1 order by created_at desc;

        
Example output

        
SELECT id, name
FROM users
WHERE active = 1
ORDER BY created_at DESC;
